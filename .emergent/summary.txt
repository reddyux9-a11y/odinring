**User Side Features:**
*   **Profile Dashboard:** Drag-and-drop link manager, visibility toggles, categorization. Redesigned to simplify creation and move extensive customization to a dedicated section.

Enhanced with custom logo upload, footer visibility toggle, 8 color theme presets, typography options, 6 link card styles, and drag-and-drop link reordering.
*   **Public Profile:** Mobile-first card layout, bold link buttons, icons, quick contacts. Now dynamically reflects all customization settings.
*   **NFC Ring Integration:** Links to default profile or Direct Link Mode.
*   **Analytics:** Profile visits, per-link clicks (simple graphs).
*   **Mobile-First Enhancements:** Bottom navigation, swipe gestures, pull-to-refresh, touch-friendly drag handles, floating action button. Mobile link view improved for full information display.
*   **QR Codes:** Generate profile/link QR codes, scan tracking.
*   **Smart Scheduling:** Availability management, appointment booking with time slots, link scheduling.
*   **Authentication:** Simple email/password. Enhanced with Remember Me feature.

**Admin Side Features:** Dashboard overview, user management, ring management (bulk CSV upload, export), analytics, customization control, support tools.
