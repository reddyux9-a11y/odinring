# 📑 Content Tabs - Quick Guide

## **What Changed**

**Before:**
- Sidebar had separate "Items" and "Links" sections

**After:**
- Single "Content" section with **2 tabs inside**

---

## **How It Works**

### **Step 1: Click "Content" in Sidebar**
Opens the Content Manager

### **Step 2: See Two Tabs**
```
[Links 🔗 (5)]  [Items 🛍️ (3)]
```

### **Step 3: Switch Between Tabs**
- **Tab 1 (Links)**: Manage all your profile links
- **Tab 2 (Items)**: Manage all your merchant items

---

## **Features**

✅ **Badge Counters**: Shows number of links/items on each tab  
✅ **Color Coding**: Blue for Links, Emerald for Items  
✅ **Quick Switching**: One click between tabs  
✅ **Same Functionality**: All features work as before  

---

## **Tab Details**

### **Tab 1: Links** 🔗
- Create new links
- Edit existing links
- Toggle visibility
- Reorder with drag & drop
- Delete links
- **Badge shows:** Number of links

### **Tab 2: Items** 🛍️
- Add products/services
- Edit item details
- Set prices & currency
- Manage stock quantity
- Upload item images
- **Badge shows:** Number of items

---

## **Quick Actions**

| Action | How To |
|--------|--------|
| Switch to Links | Click "Links" tab |
| Switch to Items | Click "Items" tab |
| Add Link | Switch to Links tab → Click "+ Add Link" |
| Add Item | Switch to Items tab → Click "+ Add Item" |
| View Count | Check badge on tab (e.g., "Links (5)") |

---

## **Benefits**

1. ✅ **Less Navigation**: No more switching sidebar sections
2. ✅ **Faster Access**: Quick tab switching
3. ✅ **Better Organization**: Related content together
4. ✅ **Visual Feedback**: Always see counts on tabs
