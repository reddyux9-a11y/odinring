# 🛍️ Merchant Items Feature - Quick Start

**Feature:** Add and manage products/services in your OdinRing profile  
**Location:** Dashboard → "Your Links" section → **Top** (above links)

---

## ✅ **What You Get**

- 🛍️ **Add Products/Services** - Showcase what you sell
- 💰 **Price Management** - Set prices in multiple currencies
- 📸 **Image Support** - Add product images
- 📦 **Stock Tracking** - Monitor inventory levels
- 👁️ **Visibility Control** - Show/hide items
- 📊 **View Counter** - Track how many people view your items
- 🔄 **Reordering** - Arrange items in any order

---

## 🚀 **How to Use**

### **1. Access the Feature**
```
http://localhost:3000/dashboard
```
- Click "Your Links" in the sidebar
- See "Merchant Items" section at the top

### **2. Add Your First Item**

Click **"Add Item"** button and fill in:

**Required:**
- **Name:** Product/service name (e.g., "Premium Widget")
- **Price:** Amount (e.g., 99.99)

**Optional:**
- **Description:** Details about your item
- **Currency:** USD, EUR, GBP, etc. (default: USD)
- **Image URL:** Link to product image
- **Category:** product/service/digital
- **Stock Quantity:** How many you have
- **SKU:** Product code

Click **"Add Item"** → Done! ✅

## ✏️ **Managing Items**

### **Edit Item**
1. Click the **edit icon** (pencil) on any item
2. Modify fields
3. Click "Update Item"

### **Delete Item**
1. Click the **trash icon** on any item
2. Confirm deletion
3. Item removed ✅

### **Toggle Visibility**
- Click the **eye icon** to show/hide items
- Hidden items won't appear on your public profile

### **Reorder Items**
- Use **up/down arrows** to change order
- Items display in the order you set
