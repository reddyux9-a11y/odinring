# =============================================================================
# DEVELOPMENT QUICK START
# =============================================================================
#
# 1. Copy this file:
#    cp env-template.txt .env
#
# 2. Update FIREBASE_PROJECT_ID with your Firebase project ID
#
# 3. Download Firebase service account JSON:
#    - Go to Firebase Console > Project Settings > Service Accounts
#    - Click "Generate new private key"
#    - Save as firebase-service-account.json in project root
#
# 4. Generate a secure JWT_SECRET:
#    openssl rand -base64 32
#
# 5. Update REACT_APP_FIREBASE_* variables from Firebase Console
#
# 6. Start development:
#    - Backend: cd backend && uvicorn server:app --reload
#    - Frontend: cd frontend && npm start
